Installation guide
==================

If you are looking for an implementation
----------------------------------------

Rocket League
^^^^^^^^^^^^^

To install the Rocket League implementation of the package with its features, run the following command::
    pip install features-rlgym[rocket-league]

If you are looking for an API
-----------------------------

If you look for a way to implement features in your RLGym environment, simply run this command::
    pip install features-rlgym[api]
